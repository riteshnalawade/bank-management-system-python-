# 🏦 Bank Management System (Python)

A Menu Driven Bank Management System built using Python and Object-Oriented Programming (OOP).
It allows users to perform core banking operations with data stored in a JSON file.

---

## 🚀 Features

* Create a new bank account
* Secure login (account number & password)
* Deposit and withdraw money
* Check account balance
* View transaction history
* Mini statement (last 5 transactions)
* Transfer money between accounts
* Generate bulk test data using Faker

---

## 🛠️ Tech Stack

* Python
* JSON (data storage)
* Faker (test data generation)

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/riteshnalawade/bank-management-system.git
```

2. Navigate to the project folder

```
cd bank-management-system
```

3. Run the program

```
python bank_management_system1.py
```

---

## 📊 Data Storage

All account details and transaction history are stored in:
